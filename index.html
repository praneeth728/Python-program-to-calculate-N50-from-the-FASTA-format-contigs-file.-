Instructions

There are two ways you can run the n50 tool:

  1. Directly run n50_python.py with contigs.fasta file

    2. This will show the output in the terminal.

    python n50_python.py contigs.fasta
  2. Use n50_tool.sh

    3. In this case, you need to set the path to your environmental path so that it can run in any directory.

    export PATH=$PATH:/path/to/n50_tool
	n50_tool.sh contigs.fasta

In the second case, you will get a file named "file_name"-n50.txt in the current working directory.
